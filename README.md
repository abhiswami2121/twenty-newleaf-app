# NewLeaf Twenty App

Twenty CRM App SDK package for NewLeaf Financial — bringing the full NewLeaf operations suite into Twenty CRM.

## Architecture

4 AI agents (GLM 5.2 + Claude) | 25 skills | 15 custom views | 4 dashboards | 30 workflows | Customer 360 | NewLeaf theme

## Agents

| Agent | Model | Skills |
|-------|-------|--------|
| Sales Assistant | GLM 5.2 | 8 |
| Recovery Specialist | GLM 5.2 | 7 |
| Dispute Officer | GLM 5.2 | 7 |
| Customer Concierge | Claude Sonnet 4.6 | 7 |

## Views

15 custom views including Pipeline Kanban, Recovery Queue, Connector Health, Active Disputes, etc.

## Dashboards

4 role-based dashboards: Admin Overview, Sales, Recovery, Dispute.

## Workflows

30 automation workflows across Lead Lifecycle, Billing, Disputes, Engagement, Operations, and Recovery.

## Deployment

```bash
npx twenty app:sync newleaf
```

## Theme

NewLeaf brand: Teal #06b6a4 on dark navy #0f172a, Inter font.
